# Bài 03 - 일상생활 (Sinh hoạt hằng ngày)

!!! abstract "Mục tiêu bài học"
    Sau bài này, bạn có thể nói mình làm gì hằng ngày, hỏi người khác làm gì và ở đâu, sử dụng đuôi câu trang trọng trong giao tiếp cơ bản.

## Liên kết nhanh

- [-ㅂ/습니다](../grammar/seumnida.md)

=== "Từ vựng"

    ## Hoạt động hằng ngày

    | Tiếng Hàn | Nghĩa |
    |---|---|
    | 먹다 | ăn |
    | 마시다 | uống |
    | 공부하다 | học |
    | 읽다 | đọc |
    | 쓰다 | viết |
    | 보다 | xem |
    | 듣다 | nghe |
    | 자다 | ngủ |

    ## Địa điểm

    | Tiếng Hàn | Nghĩa |
    |---|---|
    | 학교 | trường học |
    | 집 | nhà |
    | 도서관 | thư viện |
    | 회사 | công ty |
    | 식당 | nhà hàng |

    ## Từ để hỏi

    | Tiếng Hàn | Nghĩa |
    |---|---|
    | 누구 | ai |
    | 어디 | đâu |
    | 무엇 / 뭐 | cái gì |

=== "Ngữ pháp"

    ## 1. -ㅂ/습니다

    Đuôi câu trần thuật trang trọng.

    Ví dụ:

    - 저는 밥을 먹습니다.
    - 학교에서 공부합니다.

    ## 2. -ㅂ/습니까?

    Dùng để đặt câu hỏi trang trọng.

    Ví dụ:

    - 어디에서 공부합니까?
    - 무엇을 먹습니까?

    ## 3. 을/를

    Tiểu từ tân ngữ.

    | Sau phụ âm | Sau nguyên âm |
    |---|---|
    | 을 | 를 |

    Ví dụ:

    - 밥을 먹습니다.
    - 커피를 마십니다.

    ## 4. 에서

    Chỉ nơi diễn ra hành động.

    Ví dụ:

    - 학교에서 공부합니다.
    - 집에서 쉽니다.

=== "Mẫu câu"

    | Tiếng Hàn | Nghĩa |
    |---|---|
    | 무엇을 먹습니까? | Bạn ăn gì? |
    | 밥을 먹습니다. | Tôi ăn cơm. |
    | 어디에서 공부합니까? | Bạn học ở đâu? |
    | 학교에서 공부합니다. | Tôi học ở trường. |
    | 무엇을 읽습니까? | Bạn đọc gì? |

=== "Hội thoại"

    A: 어디에서 공부합니까?

    B: 학교에서 공부합니다.

    A: 무엇을 공부합니까?

    B: 한국어를 공부합니다.

    A: 도서관에 갑니까?

    B: 네, 갑니다.

=== "Luyện tập"

    1. Đặt 5 câu với 을/를.
    2. Đặt 5 câu với 에서.
    3. Trả lời:
       - 무엇을 먹습니까?
       - 어디에서 공부합니까?
       - 무엇을 읽습니까?

=== "Tổng kết"

    | Cấu trúc | Nghĩa |
    |---|---|
    | -ㅂ/습니다 | câu trần thuật trang trọng |
    | -ㅂ/습니까? | câu hỏi trang trọng |
    | 을/를 | tân ngữ |
    | 에서 | tại, ở |

    ## Câu cần thuộc

    1. 저는 밥을 먹습니다.
    2. 어디에서 공부합니까?
    3. 학교에서 공부합니다.
    4. 한국어를 공부합니다.
    5. 무엇을 읽습니까?

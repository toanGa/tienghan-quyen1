# Bài 05 - 하루 일과 (Công việc trong ngày)

!!! abstract "Mục tiêu bài học"
    Sau bài này, bạn có thể nói về sinh hoạt hằng ngày, hỏi giờ, kể lịch trình trong ngày và sử dụng đuôi câu `-아/어요`.

## Liên kết nhanh

- [-아/어요](../grammar/a-eoyo.md)

=== "Từ vựng"

    ## Thời gian trong ngày

    | Tiếng Hàn | Nghĩa |
    |---|---|
    | 아침 | buổi sáng |
    | 오전 | buổi sáng (AM) |
    | 점심 | buổi trưa |
    | 오후 | buổi chiều (PM) |
    | 저녁 | buổi tối |
    | 밤 | ban đêm |

    ## Hoạt động hằng ngày

    | Tiếng Hàn | Nghĩa |
    |---|---|
    | 일어나다 | thức dậy |
    | 자다 | ngủ |
    | 씻다 | rửa mặt / tắm |
    | 먹다 | ăn |
    | 마시다 | uống |
    | 공부하다 | học |
    | 일하다 | làm việc |
    | 운동하다 | tập thể dục |
    | 쉬다 | nghỉ ngơi |
    | 보다 | xem |

    ## Địa điểm

    | Tiếng Hàn | Nghĩa |
    |---|---|
    | 학교 | trường học |
    | 회사 | công ty |
    | 집 | nhà |
    | 도서관 | thư viện |
    | 식당 | nhà hàng |

=== "Ngữ pháp"

    ## 1. -아/어요

    Dùng để tạo câu lịch sự thân thiện trong giao tiếp hằng ngày.

    | Động từ | Kết quả |
    |---|---|
    | 가다 | 가요 |
    | 오다 | 와요 |
    | 먹다 | 먹어요 |
    | 마시다 | 마셔요 |
    | 공부하다 | 공부해요 |

    Ví dụ:

    - 아침에 학교에 가요.
    - 커피를 마셔요.
    - 한국어를 공부해요.

    Xem thêm: [-아/어요](../grammar/a-eoyo.md)

    ## 2. 에 가다

    Dùng để diễn tả đi đến một địa điểm.

    ```text
    Địa điểm + 에 가다
    ```

    Ví dụ:

    - 학교에 가요. → Đi đến trường.
    - 회사에 가요. → Đi đến công ty.
    - 도서관에 가요. → Đi đến thư viện.

    ## 3. 안

    Dùng để phủ định đơn giản.

    ```text
    안 + Động từ/Tính từ
    ```

    Ví dụ:

    - 커피를 안 마셔요. → Tôi không uống cà phê.
    - 학교에 안 가요. → Tôi không đi học.

=== "Mẫu câu"

    | Tiếng Hàn | Nghĩa |
    |---|---|
    | 몇 시에 일어나요? | Mấy giờ bạn thức dậy? |
    | 7시에 일어나요. | Tôi thức dậy lúc 7 giờ. |
    | 아침에 뭐 해요? | Buổi sáng bạn làm gì? |
    | 학교에 가요. | Tôi đi đến trường. |
    | 저녁에 운동해요. | Buổi tối tôi tập thể dục. |

=== "Hội thoại"

    A: 몇 시에 일어나요?

    B: 7시에 일어나요.

    A: 아침에 뭐 해요?

    B: 아침을 먹고 학교에 가요.

    A: 저녁에 뭐 해요?

    B: 한국어를 공부해요.

=== "Luyện tập"

    ## Trả lời câu hỏi

    1. 몇 시에 일어나요?
    2. 아침에 뭐 해요?
    3. 학교에 가요?
    4. 저녁에 뭐 해요?

    ## Viết về lịch trình của bạn

    Viết 5-7 câu về một ngày của bạn:

    - Mấy giờ thức dậy
    - Buổi sáng làm gì
    - Buổi chiều làm gì
    - Buổi tối làm gì

=== "Tổng kết"

    | Cấu trúc | Nghĩa |
    |---|---|
    | -아/어요 | đuôi câu lịch sự |
    | 에 가다 | đi đến |
    | 안 + V | không V |

    ## Câu cần thuộc

    1. 몇 시에 일어나요?
    2. 7시에 일어나요.
    3. 학교에 가요.
    4. 한국어를 공부해요.
    5. 커피를 안 마셔요.

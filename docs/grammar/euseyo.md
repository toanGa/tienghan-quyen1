# -(으)세요

## Ý nghĩa

Dùng để:

- Mời ai đó làm gì
- Đề nghị lịch sự
- Hướng dẫn khách hàng

## Cấu trúc

| Động từ | Kết quả |
|---|---|
| 가다 | 가세요 |
| 오다 | 오세요 |
| 먹다 | 먹으세요 |
| 마시다 | 마시세요 |
| 앉다 | 앉으세요 |
|

## Ví dụ

- 많이 드세요. → Mời ăn nhiều nhé.
- 여기 앉으세요. → Xin mời ngồi đây.
- 커피를 마시세요. → Hãy uống cà phê.
- 천천히 드세요. → Ăn từ từ nhé.

## Trong nhà hàng

- 주문하세요. → Xin gọi món.
- 많이 드세요. → Chúc ngon miệng.

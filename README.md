# Korean Basic Static Site Content

Bộ file Markdown cho giáo trình Tiếng Hàn Tổng Hợp dành cho người Việt - Sơ cấp 1.

## Cấu trúc

```text
index.md
lessons/
  bai-01.md
  ...
  bai-15.md
```

Có thể dùng với GitHub Pages, Jekyll, MkDocs, Astro, VitePress hoặc Docusaurus.

## Gợi ý dùng nhanh với GitHub Pages

1. Tạo repository mới.
2. Upload toàn bộ thư mục này.
3. Vào Settings -> Pages.
4. Chọn branch `main` và thư mục root.
5. Truy cập URL GitHub Pages sau khi build xong.
